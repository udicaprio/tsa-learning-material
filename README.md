# Thermal Swing Adsorption - Further learning material

[**Adsorption Science & Technology**](https://journals.sagepub.com/home/adt): Pages 339 to 355 include a deep explanation about TSA, reporting the principle, mathematical model formulation, analysis, and application cases. Useful to understand the basis.<p>
[**TSA report and development tool from VITO**](https://emis.vito.be/nl/tools/mapitccu/technologies/temperature-swing-adsorption): Report about TSA technology readiness level. Useful to understand the application scenario.<p>
[**TSA Integration example at TU Wien**](https://youtu.be/ib5meAW8Tss?si=95LnW8RtPcv6jLjs): YouTube video showing integration and upscale example at TUWien. Useful to understand the application scenario.<p>
[**Review on electric TSA**](https://doi.org/10.1016/j.seppur.2024.128522): Review highlighting the improvement of the TSA using electric heating, reporting advantages and limitations of the various approaches. Useful to understand the research going on in the field.<p>
[**TSA for CO2 capture**](https://www.politesi.polimi.it/retrieve/a81cb05b-4283-616b-e053-1605fe0a889a/Zanco_thesis_2015.pdf): Master thesis reporting the application and integration of thermal swing adsorption for CO<sub>2</sub> capture on power plant.
